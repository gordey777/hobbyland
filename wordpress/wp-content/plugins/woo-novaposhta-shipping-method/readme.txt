== Installation ==

1. Upload the entire 'woo-novaposhta-shipping' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Set "Shipping Destination" to "Only ship to the customer's billing address"
